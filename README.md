# StepMania

Advanced cross-platform rhythm game for home and arcade use.

## Dependencies
------------

- SDL2
- OpenGL 2.1+ / OpenGL ES 2+
- OpenAL, soon libsoundio instead.
- Lua / LuaJIT / LLVM-lua
- FreeType
- PhysicsFS
- Vorbisfile
- Theora
